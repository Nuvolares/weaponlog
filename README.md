# WeaponLog

<img width="399" height="290" alt="Image" src="https://github.com/Nuvolares/weaponlog/blob/main/New_Nav.png" />

WeaponLog is a small side project I built for myself to keep track of my firearms, range time, and training progress — and I’m sharing it here in case it’s useful to others with similar needs.

It’s a cloud-hosted web app that I run on my own server. There’s no subscription, no ads, and no commercial angle behind it.

This repository exists to explain what the project is about and to give people a place to share ideas, feedback, and bugs.  
**It does not contain the application source code.**

 **Check it out:** https://www.weaponlog.com

---

## 🎯 What WeaponLog Is

WeaponLog gives you one place to:

- Track firearms, ammunition, and accessories
- Log range sessions and ammo usage
- Look back at range time and usage over time
- Use the app from anywhere through a browser

It’s hosted and maintained by me, mainly because that’s how I prefer to run things and keep control over the setup.

---

## 🧠 Why I Built This

WeaponLog started because I couldn’t find a single tool that did *everything* I wanted.

I was looking for something that combined:
- Inventory tracking
- Range session logging
- A way to keep an eye on range time and related costs
- And eventually, insight into how my target training improves over time

There are some really good apps out there that do parts of this very well. But whenever I wanted features like **target marking, score calculation, and a clean combination of range activity and inventory**, I always ended up switching between tools.

So I decided to give it a try and build something for myself — tailored to the way I train, track, and review my progress, and running on a server I own.

WeaponLog is the result of that effort.

---

## ✨ What It Can Do

- 📦 **Inventory**
   - Firearms, ammo, and accessories
   - Purchase dates, invoices, notes, and photos
   - Track weapon rentals and associated costs
   - Membership management with renewal reminder

- 🎯 **Range Sessions**
   - Log sessions, weapons used, and ammo consumption
   - Upload and analyze target photos
   - Track shooting distance per target
   - Keep a clean history over time

- 📊 **Training Progress**
   - Mark hits directly on target photos
   - Automatic score calculation (bullseye weighting)
   - Group size measurement with MOA calculation (for distances 100+ yards)
   - Define aim zones for zone-specific accuracy tracking
   - Track improvement trends over time

- 👤 **Accounts & Security**
  - Email verification on signup
  - Secure login
  - Self-service password reset

- ☁️ **Cloud-Hosted**
  - No local install needed
  - Accessible anywhere via browser

- 🎨 **Simple, Clean UI**
  - Dark theme
  - Mobile-friendly
  - Focused on usability

---

### 🎯 Target Analysis
Upload your target photos and mark your shots directly in the browser. WeaponLog calculates:
- Hit scores with bullseye weighting
- Group size in inches or centimeters
- MOA at longer distances
- Zone-specific accuracy when using aim zones
